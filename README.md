# Mi-LIZ
Un repositorio para sorprender a Liz en cada fecha importante: aniversarios, cumpleaños, días festivos y momentos espontáneos. Aquí conviven cartas, felicitaciones, mini‑webs y experiencias interactivas hechas con HTML, CSS y JS, pensadas para emocionarla, hacerla sonreír: esperando dejar huella y me llegue a recordar.
